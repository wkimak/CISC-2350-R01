Simon Game

I built this app using vanilla javascript. The biggest challenge was timing the button animations correctly. SetInterval/SetTimeOut methods were heavily used, along with .push() and .splice() array methods to keep track of user input.

Tic-Tac-Toe

I built this app using vanilla javascript. I got a lot of practice with scope between functions and experienced how useful arrays are in storing/keeping track of data. Regex Form

This project gave me practice with regex expressions. I used the .test() method to test if input values matched the given regex expression. If .test() returned false, the description would show. Also, I used the .replace() method to format the telephone number.

Random Quotes

I built this app to practice looping through an array of quotes. I used Math.floor(Math.random)()*array.length) so the screen would output a random quote on each click event. Also, I learned how simple Jquery makes for certain animations. This project was easy to build but got me more comfortable with Jquery animations, conditionals, loops, and event listeners.

Flickr API

I built this app using Jquery's $.getJSON method, along with .$each to loop through and extract images/links in my callback function. I experimented with both jQuery and vanilla javascript, and got a better understanding of the difference between JSON and XML.

Wikipedia Viewer

Initially, the server was not allowing me to access the data with plain JSON. Although I learned that I can request data in my request URL, I prefer to make a seperate object to make the three essential steps in API .get() requests more clear for myself.
