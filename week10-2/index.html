Transition property

The transition property is a shorthand property used to represent 'transition-property', 'transition-duration', 'transition-timing-function', and 'transition-delay'.

transition-property: Define what property, or properties, you want to apply a transition effect to. You can include a comma-seperated list of property names for transitioning multiple properties on a single element. You can also use the keyword 'none' or 'all'.

transition-duration: Define the duration of a specified transition. The length of time it will take for the targeted elements to transition between two defined states. The time can be defined in either seconds(s) or milliseconds(ms).

transition-timing-function: Define a function that descirbes how a transition will proceed over its duration, allowing a transition to change speed during its course. Keywords values include 'ease', 'linear', 'ease-in', 'ease-out', 'ease-in-out', 'step-start', and 'step-end'.

transition-delay: Define a length of time to delay the start of a transition. Time can be defined in either seconds(s) or milliseconds(ms).

Example: transition: background-color 0.5s ease 1000ms;
