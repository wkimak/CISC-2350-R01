The background-blend-mode property allows a background image to blend in with its background color. Text on top of the background image is not affected. We can also use multiple linear gradients and each of them can have their own blend mode.

Values include:

  • normal - background-color will not bleed through the background-image.
  • multiply - background-image and background-color are multiplied. This leads to a darker color.
  • screen - both image and color is inverted, multiplied, and then inverted again.
  • overlay - background-color is mixed with the background-image to reflect the lightness or darkness of the backdrop
  • darken - if the background-image is darker than the background-color then the image is replaced, otherwise it is left as it was.
  • lighten - if background-image is lighter than the background-color then the image is replaced, otherwise it is left as it was
  • color-dodge - background-color is divided by the inverse of the background-image
  • color-burn - background-color is inverted, divided by the background-image and inverted again.
  • Other values include hard-light , soft-light , difference , exclusion , hue , saturation , color , and luminosity .

The transfrom property can be used with the value of rotate. If you just want to rotate an element on its Y axis, you can set the value to rotateY(..deg); You can do the same for the X axis using rotateX(..deg); You can mix in transform properties with CSS animations to add awsome affects.
